网络科技协会资料归档

　每一届协会的管理成员都有义务管理并维护整个云盘，保证云盘里文档的有效和整齐

　所有文档分为三类

　1. 活动文档

以年份命名，每一活动独立建立一个新的文件夹，是一年的活动记录，作用是新一届回顾使用，作为协会的历史积累下来

  • 网页大赛
  • 纳新
  • 学习，分享会和学习计划制定
  • 娱乐活动
  • 其他活动

    　2. 记录文档

    存放在文档里，记录活动文档中比较重要的事情，还有协会会一直使用的事情
  • 协会成员信息，每一年纳新和毕业都需要添加和更新
  • 各种账号信息，例如微信公众平台，麦客，邮箱，随手记，域名,等账户
  • 协会状况，里面有协会简介，分组简介，和每一届成员构架，协会章程(比较老了)
  • 通信稿，每一活动给学校提供的通信稿
  • 分享会PPT模板
  • LOGO
  • 杂七杂八的活动文档，当你感觉比较有用，但又无法归类时，放进去，比如说 教室申请单

    3. 协会作品

    存放协会的项目，但必须是已经完善，或是已经接近完善的， 可以选择存放简单的介绍性文档，但最好提供github链接，支持开源

注意： 1.如果是协会成员自己做的图片，psd文件最好一同放进去 2.每一活动下都需要基本的 文档 图片 基本文件夹 ，其他文件夹可添加 3.一二级文件夹的添加和修改最好在该文档里有说明，本文档采用的是Markdown语法，修改可以在 文档/协会状况和规则/协会云盘规则/一二级文件存放规则.md里实现，但需要重新生成PDF,以更换该文档 4.所有文档以2015年最切合标准，之前的部分符合
