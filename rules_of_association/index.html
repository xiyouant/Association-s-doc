网络科技协会资料归档

第一章 总则

第一条、协会名称：西安邮电大学计算机学院网络科技协会

第二条、 协会性质：本协会是一个在院团委直接领导和管理下的，严格遵守国家法律、学校纪律和协会章程的，有组织、有纪律、有素质的，以发现和培养计算机网络技术人才和组织管理人才为宗旨的学生组织。本协会的各项工作是在计算机学院领导的指导和学院分团委的管理下，由网络工程系和网络工程实验室联合组织和承担。

第三条、 协会宗旨：资源分享，共同进步，尽享网络之美。

第四条、 协会理念：分享（Share），创意（Idea），超越（Beyond）。

第五条、 协会口号：网络让我们更近，我们让网络更进。

第六条、 协会任务：协会以增强广大成员的网络科技意识，激发会员学习网络知识的兴趣，全面提高会员的网络应用、网络管理、网络安全等方面的知识和能力为宗旨，使会员在计算机网络技术领域得以全面发展，让会员在未来的社会职场中拥有更强的竞争力，积极投身于社会主义现代化的建设之中，为祖国美好的明天增光添彩。

第二章 组织机构

第7条、 成立协会常委会，下设协会会长一名，并设置四个网络技术小组，每个技术小组分别设置组长一名，学生组员若干。

第8条、 协会会长负责会议的安排和主持；指导协会各技术小组开展工作，并督促检查工作的落实情况；围绕学校和社会每个时期的工作重点，负责制定协会每个阶段的工作计划；负责向学校反应协会会员的意见和建议；处理会员的意见建议和投诉。各技术小组组长及副组长负责制定本小组的活动计划以及各项活动的组织安排工作，配合会长完成协会的各项工作 。

第九条、 协会由以下四个技术小组组成：

1、 网络应用开发组

网络应用开发组的研究方向侧重于计算机网络技术的实际应用，如网页设计，web 开发技术、基于网络的各种应用程序的开发与应用等。负责计算机学院及学校其他部门乃至校外企事业单位的网站的开发和维护工作。指导教师：苑庆涛、余浚、王亚刚、陈有为、李刚 。

2、 网络安全技术组

网络工程设计组的研究方向侧重于计算机网络环境中数据的保密性、完整性及可用性，网络的 攻击与防范等，指导会员学习和研究网络安全方面的知识，指导学生参加“全国大学生信息安全竞赛”。指导教师：刘建华、孙韩林、朱辉、王小耿、马素刚等 。

3、 技术运营组

网络工程设计组的研究方向侧重于计算机网络运维及服务器应用的开发、Linux 系统。在老师的指导下学习网络技术的前沿知识，例如云计算，大数据等。指导教师：夏虹、陈彦萍、孙韩林、王亚刚、翟社平等 。

4、 视觉设计组

视觉设计组的研究方向侧重于 WEB 视觉设计和UI交互等内容，为协会其他小组成员提供设计支持。指导教师：苑庆涛、潘春华等 。

第十条、 协会会员代表大会每学期召开一次。常委会可根据实际情况，经批准可适当提前或延迟会期。延期换届最长不得超过一年。

第3章 会员制度

第十一条、申请加入协会的会员，必须具备以下条件

  1. 有加入协会的意愿；

  2. 协会成员必须是本校学生；

  3. 承诺和遵守本章程 。

第十二条、会员入会的程序

  1. 提交入会的书面申请书；

  2. 协会常委会讨论是否批准入会；

  3. 经协会常委会批准后，由协会发放会员证 。

第十三条、会员享有以下权利：

  1. 协会会员均享有选举权、被选举权和表决权；

  2. 协会会员优先参加协会组织的各项活动；

  3. 协会会员优先享受协会提供的服务；

  4. 对协会的各项工作进行监督，并提出建议和意见；

  5. 入会自愿，退会自由 。

第十四条、会员履行以下义务：

  1. 自觉遵守协会的章程，维护协会的名誉；

  2. 认真执行协会的决议；

  3. 积极参与协会及格技术小组组织的活动；

  4. 及时完成协会交给的各项任务；

  5. 及时反应协会活动过程中遇到的各种问题，并提出自己的建议；

  6. 会员之间应该团结友爱，相互交流，共同进步 。

第十五条、会员主动退会应提交书面申请书，并交回会员证。协会会员若连续2个月不参加协会活动，视为自动退会 。

第4章 附则

第十六条、本章的修订，须经协会会员全体大会表决通过 。

第十七条、本章程最终解释权归协会常委会所有 。
